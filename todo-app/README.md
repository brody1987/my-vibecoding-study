# React Todo App

A simple Todo List application built with React + TypeScript + Vite.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persists state in memory (refresh clears data)

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown in the terminal (usually http://localhost:5173).
